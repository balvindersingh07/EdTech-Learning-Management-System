# Screenshot checklist — GitHub / repository proof

Capture **after** your final push to `main` (or your submission branch).

- [ ] **Repo home** — public URL, description, topics, commit activity graph.
- [ ] **`README.md` rendered** — scroll showing Overview, Architecture, Design documentation links.
- [ ] **Commit history** — last 5–10 commits with clear messages (design, docs, tests).
- [ ] **Pull Requests (optional)** — if you used PRs for capstone phases.
- [ ] **`docs/` folder** — GitHub file browser showing `uml/`, `patterns/`, `testing/`, `final-presentation/`.

**Optional:** Contributors graph or Insights → Network.

Save images as `docs/screenshots/github-*.png` or attach directly to your report.
